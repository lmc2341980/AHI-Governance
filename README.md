# AHI-Governance
Kho lưu trữ này quản lý các chính sách bảo mật dữ liệu và quy trình chuyển đổi từ dịch vụ công cộng sang chủ quyền cục bộ offline. Nó thiết lập các tiêu chuẩn đánh giá rủi ro hệ thống và kế hoạch quản trị thay đổi giảm thiểu sự kháng cự từ nhân sự. Mục tiêu cốt lõi là bảo tồn di sản tri thức trường tồn cho tổ chức
